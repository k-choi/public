# 오차 범위

현대 로맨스 스릴러 · 전 4부 28장.

측정할 수 없는 것에 항복하는 일 — 조작된 시험성적서를 사이에 둔, 어리숙한 신소재공학도와 그에게 접근한 요원의 이야기.

## 온라인으로 읽기

GitHub Pages가 배포되면 아래 주소에서 읽을 수 있습니다:

**https://k-choi.github.io/public/**

- `index.html` — 리더(표지·차례·전 28장, 라이트/다크)
- `manuscript/` — 장별 원고(Markdown)

> Pages 배포는 `.github/workflows/deploy-pages.yml`가 `main` 푸시 시 자동으로 활성화·배포합니다.
> 자동 활성화가 막히면 저장소 Settings → Pages → Source를 **GitHub Actions**로 한 번만 지정하세요.
